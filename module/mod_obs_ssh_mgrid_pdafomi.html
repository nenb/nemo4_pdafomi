PDAF-OMI observation module for ssh observations (on model grid)

The subroutines in this module are for the particular handling of ssh observations available on the model grid.

The routines are called by the different call-back routines of PDAF. Most of the routines are generic so that in practice only 2 routines need to be adapted for a particular data type. These are the routines for the initialization of the observation information (init_dim_obs) and for the observation operator (obs_op).

The module uses two derived data type (obs_f and obs_l), which contain all information about the full and local observations. Only variables of the type obs_f need to be initialized in this module. The variables in the type obs_l are initialized by the generic routines from PDAFomi.


Uses
